# Chelsea data
## Data sources used in Tableau (dashboard)
For our dashboard, we used the following data:  
- data/Chelsea_shapefiles/MA2024.shp    
- [chelsea_all.xlsx](https://docs.google.com/spreadsheets/d/1_t8a9_5W0-V2raIXl9bibN-321GuNcsA/edit?usp=drive_link&ouid=116575018275298979138&rtpof=true&sd=true)    
The latter is not uploaded on GitHub due to its file size (57.1 MB). The hyperlink is a Google Drive link (restricted to Boston University users as of May 9, 2025). This is a cleaned dataset that was produced from many of the other files in this folder, including raw voter and CVAP data.
## Where is the raw voter data?  
You can find them in "data/Chelse PRR_Voter Data 10 Years (2013-2023)" (early voting & absentee voting) and "data/Chelsea_in_person_data" (in-person voting).
